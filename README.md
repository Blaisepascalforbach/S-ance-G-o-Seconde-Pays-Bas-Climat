# Séance Géo Seconde — Pays-Bas & Climat

Cette publication contient une version HTML autonome de la séance interactive sur les Pays-Bas face au changement climatique.

## Contenu

- `index.html` : activité multimodale interactive prête à ouvrir dans un navigateur
- assistant de compréhension intégré : aide l'élève à reformuler, relire et débloquer son raisonnement sans fournir de réponse rédigée à sa place

## Mise en ligne

Pour publier avec GitHub Pages :

1. ouvrir les réglages du dépôt ;
2. activer GitHub Pages ;
3. choisir la branche `main` et le dossier racine `/`.

## Assistant IA

L’activité inclut un tuteur pédagogique directement dans la page.

- sur GitHub Pages, il fonctionne immédiatement en mode guidage local
- si un endpoint sécurisé compatible est configuré, il peut aussi s'appuyer sur une IA distante
- dans les deux cas, l'accompagnement est conçu pour donner des indices, poser des questions et encourager la reformulation, sans livrer de corrigé tout fait

L’activité utilise une vidéo YouTube intégrée pour rester simple à publier sur GitHub.
