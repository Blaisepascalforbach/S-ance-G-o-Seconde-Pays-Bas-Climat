# Séance Géo Seconde — Pays-Bas & Climat

Cette publication contient la version GitHub Pages de la séance interactive sur les Pays-Bas face au changement climatique.

## Contenu

- `index.html` : activité légère et autonome, directement lisible dans GitHub Pages
- tuteur de compréhension intégré : aide l'élève à reformuler, relire et débloquer son raisonnement sans fournir de réponse rédigée à sa place
- vidéo didactisée et séquencée : 4 repères guidés pour observer, noter l’idée essentielle et structurer la réponse
- parcours de synthèse : documents, exercice de remise en ordre, vidéo et rédaction finale

## Tuteur pédagogique local

L’activité inclut un tuteur pédagogique directement dans la page.

- il fonctionne sans IA générative et sans service externe
- il propose des indices, des relances, des reformulations et un guidage pas à pas
- il valorise ce que l’élève a déjà repéré avant de proposer une nouvelle piste
- il est conçu pour accompagner la compréhension sans livrer de corrigé tout fait

## Mise en ligne

Pour publier avec GitHub Pages :

1. ouvrir les réglages du dépôt ;
2. activer GitHub Pages ;
3. choisir la branche `main` et le dossier racine `/`.

La vidéo est intégrée via YouTube pour rester simple à publier sur GitHub Pages.
